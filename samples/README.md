# Test suite samples collection

This directory contains test suites description files that can be provided to the `event-generator suite` sub-commands.
These files are provided for demonstrative and testing purposes.

The following list aims at describing the content and the scope of each of them (notice that some of these files can be
used with `event-generator suite run`, but not with `event-generator suite test`):

- [syscalls.yaml](./syscalls.yaml) - contains an example for each system call supported by the `syscall` step.
